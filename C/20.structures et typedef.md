<span style="color: #3498db">20. structures et typedef</span>

Les **structures** permettent de regrouper plusieurs variables de types différents sous un seul nom. Cela rend le code plus organisé et facilite la gestion de données complexes.

---

### **1. Déclaration d’une structure**

```c id="f1k9xz"
struct Personne {
    char nom[50];
    int age;
    float taille;
};
```

* `struct Personne` = définition du type
* Contient : une chaîne (`nom`), un entier (`age`) et un flottant (`taille`)

---

### **2. Déclaration et initialisation d’une variable structure**

```c id="g2l8pm"
struct Personne p1;
strcpy(p1.nom, "Paul");
p1.age = 20;
p1.taille = 1.85;
```

* Accès aux membres avec le **point `.`**
* On peut initialiser directement à la déclaration :

```c id="h3m9qn"
struct Personne p2 = {"Marie", 25, 1.70};
```

---

### **3. typedef pour simplifier les noms**

* `typedef` permet de donner un **alias à un type** :

```c id="i4n0ro"
typedef struct Personne Personne;
Personne p3;
strcpy(p3.nom, "Luc");
p3.age = 30;
p3.taille = 1.75;
```

* Plus besoin de taper `struct Personne`, on utilise simplement `Personne`

---

### **4. Pointeurs sur structures**

```c id="j5o1sp"
Personne *ptr = &p2;
printf("%s\n", ptr->nom);   // '->' pour accéder aux membres via un pointeur
ptr->age = 26;
```

* `->` = accéder aux membres quand on utilise un pointeur

---

### **5. Tableaux de structures**

```c id="k6p2tq"
Personne groupe[3] = {
    {"Alice", 22, 1.68},
    {"Bob", 28, 1.80},
    {"Charlie", 19, 1.75}
};

for (int i = 0; i < 3; i++) {
    printf("%s a %d ans\n", groupe[i].nom, groupe[i].age);
}
```

* Permet de gérer facilement plusieurs éléments du même type structuré

---

### **6. Structures imbriquées**

```c id="l7q3vy"
struct Adresse {
    char rue[50];
    char ville[30];
};

struct Personne {
    char nom[50];
    int age;
    struct Adresse adr;
};

struct Personne p = {"Paul", 20, {"12 rue A", "Paris"}};
printf("%s habite %s\n", p.nom, p.adr.ville);
```

* Une structure peut contenir **d’autres structures**

---

### **7. À retenir**

1. Les **structures** regroupent des variables de types différents
2. `.` = accès aux membres, `->` = accès via pointeur
3. `typedef` simplifie la syntaxe
4. Utiliser **tableaux de structures** pour gérer plusieurs objets similaires
5. Les structures peuvent être **imbriquées** pour représenter des données complexes
