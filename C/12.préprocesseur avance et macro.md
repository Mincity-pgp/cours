<span style="color: #3498db">12. préprocesseur avancé et macros</span>

Le **préprocesseur** agit **avant la compilation** pour préparer le code. Les macros permettent de définir des constantes ou des fonctions simples, et d’effectuer des manipulations conditionnelles.

---

### **1. #define – définir une constante ou macro**

```c id="f1q9xz"
#define PI 3.14159       // constante
#define MAX(a,b) ((a) > (b) ? (a) : (b))  // macro fonction
```

**Explications :**

* `#define PI 3.14159` → remplace toutes les occurrences de `PI` par `3.14159` avant compilation
* `#define MAX(a,b) ...` → macro fonction : remplace MAX(a,b) par l’expression correspondante
* Les macros **n’ont pas de type** et sont **expansées textuellement**

⚠️ Attention aux parenthèses pour éviter les erreurs d’ordre d’opération.

---

### **2. #include – inclure des fichiers**

```c id="g2r8pm"
#include <stdio.h>   // bibliothèques standard
#include "monfichier.h"  // fichier utilisateur
```

* `<...>` → fichiers système
* `"..."` → fichiers du projet
* Permet de réutiliser du code ou des déclarations

---

### **3. #ifdef / #ifndef / #endif – inclusion conditionnelle**

```c id="h3s9qn"
#ifndef TEST
#define TEST
#endif
```

* `#ifndef TEST` → si `TEST` n’est pas défini
* `#define TEST` → on le définit
* `#endif` → fin du bloc
* Utile pour éviter les **inclusions multiples**

Exemple pratique :

```c id="i4t0ro"
#ifndef MA_LIB_H
#define MA_LIB_H

void maFonction();

#endif
```

* Cela évite que `maFonction()` soit définie plusieurs fois si le fichier est inclus plusieurs fois

---

### **4. #if / #elif / #else / #endif – conditions**

```c id="j5u1sp"
#define VERSION 2

#if VERSION == 1
    printf("Version 1\n");
#elif VERSION == 2
    printf("Version 2\n");
#else
    printf("Autre version\n");
#endif
```

* Permet de compiler du code différent selon les conditions
* Utile pour gérer plusieurs plateformes ou versions

---

### **5. #undef – supprimer une définition**

```c id="k6v2tq"
#undef PI
```

* Supprime la définition précédente de `PI`
* Utile pour redéfinir une macro

---

### **6. Exemple complet**

```c id="l7w3ux"
#include <stdio.h>

#define PI 3.14159
#define MAX(a,b) ((a) > (b) ? (a) : (b))

int main() {
    double r = 5.0;
    printf("Surface cercle : %.2f\n", PI * r * r);

    int x = 10, y = 20;
    printf("Le plus grand : %d\n", MAX(x,y));

    return 0;
}
```

**Résultat :**

```id="m8x4vy"
Surface cercle : 78.54
Le plus grand : 20
```

---

### **7. À retenir**

* Les directives du préprocesseur s’exécutent **avant la compilation**
* `#define` → constantes et macros
* `#include` → inclure des fichiers
* `#ifdef / #ifndef / #if` → inclusion conditionnelle
* `#undef` → supprimer une définition
* Les macros sont **pratiques mais doivent être utilisées avec précaution**
