<span style="color: #3498db">13. fichiers</span>

En C, on peut **lire et écrire des données dans des fichiers** pour stocker des informations de manière permanente.

---

### **1. Ouverture d’un fichier**

```c id="f1k9xz"
FILE *fichier;
fichier = fopen("monfichier.txt", "w");  // ouverture en écriture
```

**Explications :**

* `FILE *fichier` → pointeur vers un fichier
* `fopen("nom", "mode")` → ouvre le fichier
* `"w"` → écriture (crée le fichier ou écrase s’il existe)
* `"r"` → lecture
* `"a"` → ajout (append)

⚠️ Toujours vérifier si le fichier a été ouvert correctement :

```c id="g2l8pm"
if (fichier == NULL) {
    printf("Erreur lors de l'ouverture du fichier\n");
    return 1;
}
```

---

### **2. Écrire dans un fichier**

```c id="h3m9qn"
fprintf(fichier, "Bonjour %s\n", "Alice");
```

* `fprintf` → fonctionne comme `printf` mais écrit dans un fichier
* `%s` → format pour chaîne de caractères

---

### **3. Lire depuis un fichier**

```c id="i4n0ro"
char nom[50];
fscanf(fichier, "%s", nom);
printf("Lu depuis le fichier : %s\n", nom);
```

* `fscanf` → lit depuis le fichier comme `scanf`
* On peut lire des entiers, des flottants, ou des chaînes

---

### **4. Fermeture d’un fichier**

```c id="j5o1sp"
fclose(fichier);
```

* Toujours fermer le fichier après usage pour libérer la mémoire et sauvegarder les données

---

### **5. Exemple complet : écriture et lecture**

```c id="k6p2tq"
#include <stdio.h>

int main() {
    FILE *fichier;

    // ouverture en écriture
    fichier = fopen("monfichier.txt", "w");
    if (fichier == NULL) {
        printf("Erreur d'ouverture\n");
        return 1;
    }

    fprintf(fichier, "Bonjour Alice\n");
    fclose(fichier);

    // ouverture en lecture
    fichier = fopen("monfichier.txt", "r");
    if (fichier == NULL) {
        printf("Erreur d'ouverture\n");
        return 1;
    }

    char texte[50];
    fscanf(fichier, "%s", texte);
    printf("Lecture du fichier : %s\n", texte);

    fclose(fichier);
    return 0;
}
```

**Résultat :**

```
Lecture du fichier : Bonjour
```

⚠️ Note : `fscanf` lit **jusqu’au premier espace**, donc pour lire toute la ligne, on utilisera `fgets`.

---

### **6. Résumé rapide**

* `fopen("nom", "mode")` → ouvrir un fichier
* `fprintf` / `fscanf` → écrire / lire
* `fclose` → fermer le fichier
* `"r"` → lecture, `"w"` → écriture, `"a"` → ajout
