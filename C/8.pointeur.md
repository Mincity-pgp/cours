<span style="color: #3498db">8. pointeurs</span>

Les pointeurs sont des variables qui **contiennent l’adresse mémoire d’une autre variable**. Ils sont essentiels pour manipuler la mémoire, passer des valeurs par référence et gérer des tableaux dynamiques.

---

### **1. Déclaration d’un pointeur**

```c id="p1k9vz"
int a = 10;      // variable normale
int *ptr = &a;   // pointeur vers a
```

**Explications :**

* `int *ptr` → déclaration d’un pointeur vers un entier
* `&a` → opérateur **adresse**, donne l’adresse mémoire de `a`
* `ptr` contient maintenant l’adresse où `a` est stocké

---

### **2. Accéder à la valeur pointée (déréférencement)**

```c id="q3l8xh"
printf("Valeur de a : %d\n", *ptr);
```

* `*ptr` → opérateur de **déréférencement**, permet de lire ou modifier la valeur pointée
* Ici, `*ptr` = 10 (valeur de `a`)

On peut aussi modifier `a` via le pointeur :

```c id="r4m9yb"
*ptr = 20;
printf("Nouvelle valeur de a : %d\n", a);
```

* Résultat : `a` devient 20

---

### **3. Pointeurs et scanf**

```c id="s5n0tz"
int age;
scanf("%d", &age);   // on passe l'adresse pour stocker la valeur
```

* `&age` → on donne l’adresse de `age` pour que `scanf` écrive directement dedans

⚠️ Le `&` est **obligatoire** avec les variables normales, sauf si c’est déjà un pointeur.

---

### **4. Pointeurs et tableaux**

```c id="t6o2xl"
int tab[3] = {1, 2, 3};
int *p = tab;    // équivalent à &tab[0]

printf("%d\n", *(p + 1));  // affiche 2
```

* Les tableaux sont des pointeurs vers leur première case
* `p + 1` → adresse de la deuxième case
* `*(p + 1)` → valeur de la deuxième case

---

### **5. Pointeurs et fonctions**

Passer un pointeur à une fonction permet de **modifier la variable originale** :

```c id="u7p3kh"
void incrementer(int *val) {
    (*val)++;
}

int main() {
    int x = 5;
    incrementer(&x);
    printf("%d\n", x);  // affiche 6
    return 0;
}
```

* On passe l’adresse de `x` (`&x`)
* La fonction incrémente directement la valeur stockée à cette adresse

---

### **6. Résumé rapide**

* `*` → déréférencement (accéder à la valeur pointée)
* `&` → adresse mémoire (donner l’emplacement)
* Les pointeurs sont utilisés pour :

  * Manipuler la mémoire
  * Passer des paramètres par référence
  * Travailler avec des tableaux et structures complexes
