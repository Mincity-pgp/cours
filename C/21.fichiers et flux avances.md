<span style="color: #3498db">21. fichiers et flux avancés</span>

En C, les fichiers permettent de **stocker et récupérer des données** de manière persistante. Les flux (`FILE*`) servent à lire et écrire dans ces fichiers.

---

### **1. Ouverture et fermeture de fichiers**

```c id="f1k9xz"
#include <stdio.h>

FILE *f = fopen("exemple.txt", "w"); // ouverture en écriture
if (f == NULL) {
    printf("Erreur lors de l'ouverture du fichier\n");
    return 1;
}

// écrire dans le fichier
fprintf(f, "Bonjour le monde\n");

// fermer le fichier
fclose(f);
```

* Modes courants :

  * `"r"` → lecture
  * `"w"` → écriture (écrase le fichier existant)
  * `"a"` → ajout à la fin du fichier
  * `"r+"` → lecture/écriture

---

### **2. Lecture depuis un fichier**

```c id="g2l8pm"
FILE *f = fopen("exemple.txt", "r");
char ligne[100];

if (f != NULL) {
    while (fgets(ligne, sizeof(ligne), f) != NULL) {
        printf("%s", ligne);
    }
    fclose(f);
}
```

* `fgets(buffer, taille, fichier)` → lit une ligne
* Toujours vérifier que le fichier est **ouvert correctement**

---

### **3. Lecture et écriture formatées**

```c id="h3m9qn"
int age;
FILE *f = fopen("data.txt", "w");
fprintf(f, "%s %d\n", "Paul", 20);
fclose(f);

f = fopen("data.txt", "r");
char nom[50];
fscanf(f, "%s %d", nom, &age);
printf("%s a %d ans\n", nom, age);
fclose(f);
```

* `fprintf` / `fscanf` → comme `printf` / `scanf` mais pour les fichiers

---

### **4. Fonctions utiles pour les fichiers**

* `fseek(fichier, offset, origine)` → se déplacer dans le fichier
* `ftell(fichier)` → position actuelle dans le fichier
* `rewind(fichier)` → revenir au début du fichier
* `feof(fichier)` → vérifie si la fin du fichier est atteinte

```c id="i4n0ro"
fseek(f, 0, SEEK_END); // aller à la fin
long taille = ftell(f); // taille du fichier
rewind(f);              // revenir au début
```

---

### **5. Fichiers binaires**

```c id="j5o1sp"
struct Personne {
    char nom[50];
    int age;
};

struct Personne p = {"Paul", 20};
FILE *f = fopen("perso.bin", "wb"); // écriture binaire
fwrite(&p, sizeof(struct Personne), 1, f);
fclose(f);

struct Personne p2;
f = fopen("perso.bin", "rb"); // lecture binaire
fread(&p2, sizeof(struct Personne), 1, f);
fclose(f);
printf("%s a %d ans\n", p2.nom, p2.age);
```

* `"wb"` / `"rb"` → écriture / lecture binaire
* `fwrite` / `fread` → manipuler directement des blocs de mémoire

---

### **6. À retenir**

1. Ouvrir un fichier avec `fopen` et fermer avec `fclose`
2. Lire/écrire du texte avec `fgets`, `fputs`, `fprintf`, `fscanf`
3. Lire/écrire du binaire avec `fread`, `fwrite`
4. Vérifier toujours si le fichier a été ouvert correctement
5. Utiliser `fseek`, `ftell`, `rewind` pour manipuler la position dans le fichier
