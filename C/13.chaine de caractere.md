<span style="color: #3498db">13. chaînes de caractères</span>

Une **chaîne de caractères** est un **tableau de caractères terminé par un caractère nul `\0`**. Elle sert à stocker du texte.

---

### **1. Déclaration d’une chaîne**

```c id="f1k9xz"
char nom[] = "Paul";   // tableau de 5 caractères : 'P', 'a', 'u', 'l', '\0'
```

**Explications :**

* `"Paul"` → la chaîne se termine automatiquement par `\0`
* `char nom[5]` serait équivalent, mais il faut respecter la taille

---

### **2. Accéder aux caractères**

```c id="g2l8pm"
printf("%c\n", nom[0]); // affiche 'P'
nom[1] = 'a';           // modification d'un caractère
```

* Les chaînes sont **indexées à partir de 0**
* Chaque caractère peut être lu ou modifié

---

### **3. Fonctions de `string.h`**

C fournit une bibliothèque pour manipuler les chaînes :

```c id="h3m9qn"
#include <string.h>

char nom1[] = "Paul";
char nom2[20];

strcpy(nom2, nom1);        // copie nom1 dans nom2
strcat(nom2, " Dupont");   // concatène " Dupont" à nom2
int len = strlen(nom2);    // longueur de la chaîne (sans \0)
int cmp = strcmp(nom1, nom2); // compare deux chaînes : 0 si identiques
```

* `strcpy(dest, src)` → copie une chaîne
* `strcat(dest, src)` → ajoute une chaîne à la fin
* `strlen(str)` → longueur de la chaîne
* `strcmp(str1, str2)` → compare deux chaînes

---

### **4. Lecture de chaînes**

```c id="i4n0ro"
char nom[50];
scanf("%s", nom);       // lit jusqu'au premier espace
fgets(nom, 50, stdin);  // lit toute la ligne, y compris les espaces
```

* `scanf` → lit un mot
* `fgets` → lit une ligne complète et garde le `\n` si présent

---

### **5. Exemple complet**

```c id="j5o1sp"
#include <stdio.h>
#include <string.h>

int main() {
    char nom[50] = "Paul";

    printf("Nom initial : %s\n", nom);

    // modification
    nom[0] = 'J';
    printf("Nom modifié : %s\n", nom);

    // copie et concaténation
    char nom2[50];
    strcpy(nom2, nom);
    strcat(nom2, " Dupont");
    printf("Nom complet : %s\n", nom2);

    // longueur
    printf("Longueur : %d\n", (int)strlen(nom2));

    return 0;
}
```

**Résultat :**

```id="k6p2tq"
Nom initial : Paul
Nom modifié : Jaul
Nom complet : Jaul Dupont
Longueur : 11
```

---

### **6. À retenir**

* Les chaînes sont des **tableaux de `char` terminés par `\0`**
* `string.h` contient toutes les fonctions utiles pour copier, concaténer, comparer, et mesurer les chaînes
* Pour lire une chaîne avec espaces, **préférer `fgets`**
* Attention aux dépassements de mémoire lors de la manipulation de chaînes
