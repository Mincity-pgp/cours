<span style="color: #3498db">18. bibliothèques standard avancées</span>

Les bibliothèques standard en C contiennent des **fonctions prêtes à l’emploi** pour manipuler les chaînes, les nombres, les dates et plus encore.

---

### **1. string.h – manipulation de chaînes**

```c id="f1k9xz"
#include <string.h>

char str1[20] = "Bonjour";
char str2[20];

// copier une chaîne
strcpy(str2, str1);           // str2 = "Bonjour"

// concaténer
strcat(str2, " le monde");     // str2 = "Bonjour le monde"

// longueur
int len = strlen(str2);        // len = 17

// comparaison
int cmp = strcmp(str1, str2);  // cmp != 0
```

**Fonctions principales :**

* `strcpy(dest, src)` → copie une chaîne
* `strcat(dest, src)` → concatène des chaînes
* `strlen(str)` → retourne la longueur
* `strcmp(str1, str2)` → compare deux chaînes

---

### **2. math.h – fonctions mathématiques**

```c id="g2l8pm"
#include <math.h>

double x = 9.0;
double racine = sqrt(x);    // racine carrée
double puissance = pow(2,3); // 2^3 = 8
double valeurAbsolue = fabs(-5.5);
double cosinus = cos(3.14159 / 2);
```

**Fonctions principales :**

* `sqrt(x)` → racine carrée
* `pow(x, y)` → x puissance y
* `fabs(x)` → valeur absolue
* `sin(x), cos(x), tan(x)` → fonctions trigonométriques

---

### **3. stdlib.h – utilitaires généraux**

```c id="h3m9qn"
#include <stdlib.h>

int n = rand() % 100;   // nombre aléatoire entre 0 et 99
qsort(array, n, sizeof(int), compare); // tri d’un tableau
```

* `rand()` → nombre aléatoire
* `srand(time(NULL))` → initialise le générateur aléatoire
* `qsort()` → tri rapide d’un tableau

---

### **4. time.h – gestion du temps**

```c id="i4n0ro"
#include <time.h>

time_t now = time(NULL);       // temps actuel en secondes depuis 1970
struct tm *local = localtime(&now); 
printf("Année : %d\n", local->tm_year + 1900);
printf("Mois : %d\n", local->tm_mon + 1);
```

* `time(NULL)` → récupère le temps actuel
* `localtime(&time)` → convertit en heure locale
* `strftime()` → formater la date/heure en chaîne

---

### **5. stdio.h – entrée/sortie avancée**

```c id="j5o1sp"
#include <stdio.h>

FILE *f = fopen("fichier.txt", "w");  // ouverture pour écriture
fprintf(f, "Hello world\n");          // écrire dans le fichier
fclose(f);                             // fermer le fichier
```

* `fopen()` → ouvrir un fichier
* `fprintf()` → écrire formaté dans un fichier
* `fscanf()` → lire formaté depuis un fichier
* Toujours **fermer les fichiers** après utilisation

---

### **6. Exemple complet**

```c id="k6p2tq"
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <math.h>
#include <time.h>

int main() {
    // chaîne
    char str[50];
    strcpy(str, "Bonjour");
    strcat(str, " le monde");
    printf("%s\n", str);

    // math
    double r = 5.0;
    printf("Surface cercle : %.2f\n", M_PI * pow(r,2));

    // aléatoire
    srand(time(NULL));
    int n = rand() % 100;
    printf("Nombre aléatoire : %d\n", n);

    // date
    time_t now = time(NULL);
    struct tm *local = localtime(&now);
    printf("Date : %d/%d/%d\n", local->tm_mday, local->tm_mon+1, local->tm_year+1900);

    return 0;
}
```

**Résultat possible :**

```
Bonjour le monde
Surface cercle : 78.54
Nombre aléatoire : 42
Date : 5/4/2026
```
