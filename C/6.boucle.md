<span style="color: #3498db">6. boucles</span>

Les boucles permettent de **répéter un bloc de code plusieurs fois** tant qu’une condition est remplie.

---

### **1. while**

```c id="n4v7kq"
while (a < 10) {
    a++;
}
```

**Explications :**

* `while (a < 10)` → tant que `a` est inférieur à 10, le bloc sera exécuté
* `{ ... }` → bloc de code à répéter
* `a++` → incrémente `a` à chaque passage
* Si `a >= 10` dès le début, la boucle **ne s’exécute pas**

⚠️ Attention aux **boucles infinies** :

```c id="p5q2lr"
while(1) {
    printf("infini\n");
}
```

Cette boucle ne s’arrête jamais.

---

### **2. for**

```c id="t9k4mz"
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}
```

**Explications :**

* `int i = 0` → initialise la variable de contrôle
* `i < 10` → condition : tant qu’elle est vraie, la boucle continue
* `i++` → action après chaque itération (ici, incrémenter `i`)
* `{ ... }` → bloc à répéter
* Affiche les nombres de 0 à 9

---

### **Différences principales :**

| Boucle  | Quand l’utiliser                             | Particularité                                                          |
| ------- | -------------------------------------------- | ---------------------------------------------------------------------- |
| `while` | quand on ne sait pas combien de fois répéter | teste la condition **avant** le bloc                                   |
| `for`   | quand on sait combien de fois répéter        | intègre initialisation, condition et incrémentation dans la même ligne |

---

### **Exemple combiné :**

```c id="b2l7yf"
#include <stdio.h>

int main() {
    int a = 0;

    // boucle while
    while (a < 5) {
        printf("while : %d\n", a);
        a++;
    }

    // boucle for
    for (int i = 0; i < 5; i++) {
        printf("for : %d\n", i);
    }

    return 0;
}
```

**Résultat :**

```
while : 0
while : 1
while : 2
while : 3
while : 4
for : 0
for : 1
for : 2
for : 3
for : 4
```
