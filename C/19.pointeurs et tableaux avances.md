<span style="color: #3498db">19. pointeurs et tableaux avancés</span>

Les pointeurs sont des variables qui **contiennent l’adresse mémoire** d’une autre variable. Ils sont fondamentaux pour manipuler des tableaux, des chaînes, et la mémoire dynamique.

---

### **1. Déclaration et utilisation des pointeurs**

```c id="f1k9xz"
int a = 10;
int *p = &a;       // p pointe vers a
printf("Valeur de a : %d\n", *p); // déférencement → affiche 10
```

**Explications :**

* `int *p` → déclaration d’un pointeur vers un entier
* `&a` → adresse de la variable `a`
* `*p` → accéder ou modifier la valeur pointée

```c id="g2l8pm"
*p = 20;          // change la valeur de a
printf("%d\n", a); // affiche 20
```

---

### **2. Pointeurs et tableaux**

* Le nom d’un tableau est l’adresse de son premier élément :

```c id="h3m9qn"
int tab[5] = {1,2,3,4,5};
int *p = tab;

for (int i = 0; i < 5; i++) {
    printf("%d ", *(p+i));  // accès via pointeur
}
```

* `tab[i]` est équivalent à `*(tab + i)`

---

### **3. Tableaux de pointeurs**

* Utiles pour stocker plusieurs chaînes :

```c id="i4n0ro"
char *noms[] = {"Paul", "Marie", "Luc"};
for (int i = 0; i < 3; i++) {
    printf("%s\n", noms[i]);
}
```

* `noms[i]` est un pointeur vers une chaîne

---

### **4. Pointeurs et fonctions**

* Passer des variables par **adresse** pour les modifier :

```c id="j5o1sp"
void incrementer(int *x) {
    (*x)++;
}

int main() {
    int a = 5;
    incrementer(&a);
    printf("%d\n", a); // 6
    return 0;
}
```

* Les pointeurs permettent aussi de **passer des tableaux** à des fonctions :

```c id="k6p2tq"
void afficherTableau(int *tab, int taille) {
    for (int i = 0; i < taille; i++) {
        printf("%d ", tab[i]);
    }
    printf("\n");
}

int main() {
    int tab[5] = {1,2,3,4,5};
    afficherTableau(tab, 5);
}
```

---

### **5. Pointeurs et mémoire dynamique**

```c id="l7q3vy"
int *tab = malloc(5 * sizeof(int)); // allocation dynamique
for (int i = 0; i < 5; i++) {
    *(tab+i) = i*2;
}
free(tab); // libération mémoire
```

* `malloc` retourne un **pointeur** vers le bloc mémoire alloué
* Utiliser `*` ou la notation `tab[i]` pour accéder aux valeurs

---

### **6. Pointeurs multiples et pointeurs vers pointeurs**

* Pointeur vers pointeur :

```c id="m8r4wx"
int a = 10;
int *p = &a;
int **pp = &p;

printf("%d\n", **pp); // 10
```

* Utile pour manipuler des **tableaux de chaînes** ou des **tableaux dynamiques 2D**

---

### **7. Tableaux dynamiques 2D**

```c id="n9s5yz"
int lignes = 3, colonnes = 4;
int **matrice = malloc(lignes * sizeof(int*));
for (int i = 0; i < lignes; i++) {
    matrice[i] = malloc(colonnes * sizeof(int));
}

// remplir la matrice
for (int i = 0; i < lignes; i++)
    for (int j = 0; j < colonnes; j++)
        matrice[i][j] = i+j;

// libération
for (int i = 0; i < lignes; i++) free(matrice[i]);
free(matrice);
```

* Chaque ligne est un tableau dynamique
* Nécessite de **libérer chaque ligne puis le tableau principal**

---

### **8. À retenir**

1. Pointeurs = adresse mémoire
2. `*` = déférencement, `&` = adresse
3. Nom de tableau = adresse du premier élément
4. Passer tableaux et variables par pointeur pour modifier la valeur
5. Pointeurs essentiels pour **mémoire dynamique** et **tableaux 2D**
6. Toujours libérer la mémoire allouée
