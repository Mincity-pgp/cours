<span style="color: #3498db">9. structures</span>

Les **structures** permettent de regrouper plusieurs variables de types différents sous un même nom. Elles servent à représenter des objets ou des entités complexes dans un programme.

---

### **1. Déclaration d’une structure**

```c id="s1p9qv"
struct Personne {
    char nom[50];
    int age;
    float taille;
};
```

**Explications :**

* `struct Personne` → nom de la structure
* `{ ... }` → liste des membres (variables contenues dans la structure)
* Chaque membre peut être d’un type différent (`char`, `int`, `float`, etc.)

---

### **2. Déclaration et initialisation d’une variable structure**

```c id="t2r8mh"
struct Personne p1;

p1.age = 20;
strcpy(p1.nom, "Alice");  // copier une chaîne de caractères
p1.taille = 1.75;
```

* `p1` → variable de type `struct Personne`
* `p1.age` / `p1.nom` / `p1.taille` → accès aux membres de la structure

---

### **3. Initialisation lors de la déclaration**

```c id="u3s7kx"
struct Personne p2 = {"Bob", 25, 1.80};
```

* On peut assigner directement toutes les valeurs dans l’ordre des membres

---

### **4. Structures et fonctions**

On peut passer une structure à une fonction **par valeur** ou **par pointeur**.

**Par valeur :**

```c id="v4t8lm"
void afficherPersonne(struct Personne p) {
    printf("Nom : %s\nAge : %d\nTaille : %.2f\n", p.nom, p.age, p.taille);
}
```

**Par pointeur (plus efficace pour les grandes structures) :**

```c id="w5u9ny"
void modifierAge(struct Personne *p, int nouvelAge) {
    p->age = nouvelAge;  // opérateur -> pour accéder aux membres via pointeur
}
```

* `p->age` équivaut à `(*p).age`
* Permet de modifier directement la variable originale

---

### **5. Exemple complet**

```c id="x6v0oz"
#include <stdio.h>
#include <string.h>

struct Personne {
    char nom[50];
    int age;
    float taille;
};

void afficherPersonne(struct Personne p) {
    printf("Nom : %s\nAge : %d\nTaille : %.2f\n", p.nom, p.age, p.taille);
}

void anniversaire(struct Personne *p) {
    p->age++;
}

int main() {
    struct Personne p1 = {"Alice", 20, 1.75};

    afficherPersonne(p1);
    anniversaire(&p1);
    printf("Après anniversaire :\n");
    afficherPersonne(p1);

    return 0;
}
```

**Résultat :**

```id="y7w1px"
Nom : Alice
Age : 20
Taille : 1.75
Après anniversaire :
Nom : Alice
Age : 21
Taille : 1.75
```

---

### **6. À retenir**

* Une structure regroupe plusieurs variables sous un même nom
* On accède aux membres avec `.` ou `->` (si pointeur)
* Pratique pour représenter des objets ou des entités complexes
* Peut être utilisée avec des fonctions pour passer ou modifier des données
