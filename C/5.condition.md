<span style="color: #3498db">5. conditions</span>

Les conditions permettent de prendre des décisions dans un programme, en exécutant certaines instructions seulement si une condition est vraie.

---

### **1. if / else**

```c id="qz5n8k"
if (a > 10) {
    printf("grand");
} else {
    printf("petit");
}
```

**Explications :**

* `if (a > 10)` → teste si `a` est supérieur à 10
* `{ ... }` → bloc de code exécuté si la condition est vraie
* `else` → bloc de code exécuté si la condition est fausse
* `printf("grand")` / `printf("petit")` → affichage selon le résultat

⚠️ On peut ajouter `else if` pour tester plusieurs conditions :

```c id="s4l9pf"
if (a > 10) {
    printf("grand");
} else if (a == 10) {
    printf("égal");
} else {
    printf("petit");
}
```

---

### **2. switch / case**

```c id="k1j8wm"
switch(a) {
    case 1:
        printf("un");
        break;
    case 2:
        printf("deux");
        break;
    default:
        printf("autre");
}
```

**Explications :**

* `switch(a)` → teste la valeur de `a`
* `case X:` → exécute le bloc correspondant à la valeur `X`
* `break;` → termine le `case` et sort du `switch`
* `default:` → bloc exécuté si aucune valeur ne correspond

---

**Différences principales :**

* `if / else` → plus flexible, permet des conditions complexes (`>`, `<`, `!=`, etc.)
* `switch` → plus clair pour comparer une variable à plusieurs valeurs précises

---

**Exemple combiné :**

```c id="m8t3kj"
int a = 2;

if (a > 5) {
    printf("grand\n");
} else {
    switch(a) {
        case 1:
            printf("un\n");
            break;
        case 2:
            printf("deux\n");
            break;
        default:
            printf("autre\n");
    }
}
```

* Si `a = 2`, le programme affichera :

```
deux
```
