<span style="color: #3498db">7. fonctions</span>

Les fonctions permettent de **regrouper un bloc de code** pour le réutiliser plusieurs fois, améliorer la lisibilité et structurer le programme.

---

### **1. Déclaration et définition**

```c id="f4k9qv"
int addition(int a, int b) {   // définition de la fonction
    return a + b;              // retourne le résultat
}
```

**Explications :**

* `int addition` → le type de retour de la fonction (`int` ici)
* `(int a, int b)` → paramètres d’entrée, variables locales à la fonction
* `{ ... }` → bloc de code exécuté lors de l’appel
* `return` → renvoie une valeur à l’endroit où la fonction a été appelée

---

### **2. Appel d’une fonction**

```c id="s8l3yz"
int resultat = addition(5, 3);
printf("Résultat : %d\n", resultat);
```

* `addition(5, 3)` → appel de la fonction avec `5` et `3` comme arguments
* La fonction retourne `8`, qui est stocké dans `resultat`
* `printf` affiche la valeur retournée

---

### **3. Fonctions sans retour (`void`)**

```c id="k2q5mh"
void afficherBonjour() {
    printf("Bonjour !\n");
}
```

* `void` → indique que la fonction **ne retourne rien**
* On peut l’appeler simplement :

```c id="n6j7tl"
afficherBonjour();
```

---

### **4. Avantages des fonctions**

1. **Réutilisables** → on peut appeler la même fonction plusieurs fois
2. **Lisibles** → code plus organisé et compréhensible
3. **Paramétrables** → on peut passer des valeurs différentes à chaque appel

---

### **5. Exemple complet**

```c id="m7p4xh"
#include <stdio.h>

// fonction avec retour
int addition(int a, int b) {
    return a + b;
}

// fonction sans retour
void afficherBonjour() {
    printf("Bonjour !\n");
}

int main() {
    afficherBonjour();

    int resultat = addition(10, 15);
    printf("Résultat : %d\n", resultat);

    return 0;
}
```

**Résultat :**

```id="q8x3mv"
Bonjour !
Résultat : 25
```
