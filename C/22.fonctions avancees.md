<span style="color: #3498db">22. fonctions avancées et passage par référence</span>

Les fonctions permettent de **structurer le code**, de le rendre réutilisable et plus lisible. Le passage par référence est une technique pour **modifier les valeurs des variables en dehors de la fonction**.

---

### **1. Déclaration et définition d’une fonction**

```c id="f1k9xz"
int addition(int a, int b) {
    return a + b;
}

int main() {
    int resultat = addition(5, 3);
    printf("Résultat : %d\n", resultat);
    return 0;
}
```

* `int addition(int a, int b)` → fonction qui prend deux entiers et retourne un entier
* `return` → retourne la valeur au point d’appel

---

### **2. Passage par valeur vs passage par référence**

* **Par valeur** : la fonction reçoit **une copie** de la variable

```c id="g2l8pm"
void ajouterUn(int x) {
    x++;
}

int main() {
    int a = 5;
    ajouterUn(a);
    printf("%d\n", a); // 5 → pas modifié
}
```

* **Par référence** : la fonction reçoit l’**adresse** de la variable

```c id="h3m9qn"
void ajouterUn(int *x) {
    (*x)++;
}

int main() {
    int a = 5;
    ajouterUn(&a);
    printf("%d\n", a); // 6 → modifié
}
```

* `*x` → déférencement pour modifier la valeur
* `&a` → adresse de la variable `a`

---

### **3. Fonctions avec tableaux et chaînes**

* Les tableaux sont toujours passés **par référence** :

```c id="i4n0ro"
void doublerTableau(int tab[], int taille) {
    for (int i = 0; i < taille; i++) {
        tab[i] *= 2;
    }
}

int main() {
    int t[5] = {1,2,3,4,5};
    doublerTableau(t, 5);
    for (int i = 0; i < 5; i++) printf("%d ", t[i]); // 2 4 6 8 10
}
```

* Les chaînes (`char[]`) se comportent comme des tableaux → modifiables

---

### **4. Fonctions avec pointeurs multiples**

```c id="j5o1sp"
void swap(int *a, int *b) {
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

int main() {
    int x = 5, y = 10;
    swap(&x, &y);
    printf("x = %d, y = %d\n", x, y); // x=10, y=5
}
```

* Utile pour **échanger des valeurs**, trier des tableaux, etc.

---

### **5. Fonctions récursives**

* Une fonction peut s’appeler **elle-même** pour résoudre un problème

```c id="k6p2tq"
int factorielle(int n) {
    if (n <= 1) return 1;
    else return n * factorielle(n-1);
}

int main() {
    printf("%d\n", factorielle(5)); // 120
}
```

* Toujours prévoir une **condition d’arrêt** pour éviter les boucles infinies

---

### **6. Fonctions avec valeur de retour multiple (via pointeurs)**

```c id="l7q3vy"
void calcul(int a, int b, int *somme, int *produit) {
    *somme = a + b;
    *produit = a * b;
}

int main() {
    int s, p;
    calcul(5, 3, &s, &p);
    printf("Somme=%d, Produit=%d\n", s, p);
}
```

* Utilise des pointeurs pour **retourner plusieurs valeurs**

---

### **7. À retenir**

1. Passage par valeur → copie, ne modifie pas l’original
2. Passage par référence → adresse, permet de modifier la variable
3. Tableaux et chaînes sont passés par référence
4. Utiliser les fonctions récursives avec une condition d’arrêt
5. Les pointeurs permettent de **retourner plusieurs valeurs**
